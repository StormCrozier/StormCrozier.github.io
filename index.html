Web Scraping

This is a small script that I made while experimenting with python and web scraping. I have used python before but never attempted web scraping. For this small project, I used a python library called beautiful soup. This library has a beautiful soup object which takes the HTML from a web page and parses it into an object. That object can then be parsed by using HTML tags to get the information you are looking for. The information I was looking for was average covid cases for a specific area, the area I was using was Hawaii. The information was printed to the customer and also saved in an excel worksheet with the day and all the averages for that day.
