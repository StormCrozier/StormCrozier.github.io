Projects

Restaurant Organizer

This was a project for an object-oriented class. This is one of my first times using Java; I have experimented with java before this project but nothing as big as this. This program read in from an external file of restaurants, and these restaurants could be real or fake. The restaurants are stored in a binary search tree, and each restaurant has a liner-linked list. These linear linked lists are filled with menu items from that restaurant. This project used a lot of object-oriented designs like polymorphism and dynamic binding to name a few.

Weekly Calender

This is a project that was an assignment for an object-oriented class. It used an array to represent the days of the week and a doubly-linked list to represent activities to do for that day. This was a relatively simple project and was my first time using an inheritance hierarchy. This program was helpful for planning out some of my days, and I might expand on it in the future.

Web Scraping

This is a small script that I made while experimenting with python and web scraping. I have used python before but never attempted web scraping. For this small project, I used a python library called beautiful soup. This library has a beautiful soup object which takes the HTML from a web page and parses it into an object. That object can then be parsed by using HTML tags to get the information you are looking for. The information I was looking for was average covid cases for a specific area, the area I was using was Hawaii. The information was printed to the customer and also saved in an excel worksheet with the day and all the averages for that day.
